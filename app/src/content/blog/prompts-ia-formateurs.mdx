L'IA générative peut transformer le quotidien d'un formateur — à condition de savoir comment lui parler. Un bon prompt (une bonne instruction) fait toute la différence entre une réponse générique et un contenu immédiatement utilisable.

Voici les 10 prompts les plus utiles pour les formateurs, testés et affinés en contexte réel.

## Comment utiliser ces prompts

Ces prompts sont conçus pour Claude (claude.ai) ou ChatGPT. Copiez-collez-les et adaptez les parties en **[crochets]** à votre situation. Pensez à préciser votre contexte (type de formation, niveau du public, durée) pour obtenir des résultats plus pertinents.

---

## Prompt 1 : Créer un plan de formation complet

```
Tu es un ingénieur pédagogique expert. Crée un plan de formation complet sur le thème [THÈME] pour un public de [PROFIL DES APPRENANTS] avec un niveau [DÉBUTANT/INTERMÉDIAIRE/AVANCÉ].

La formation dure [DURÉE]. Format : présentiel / distanciel / blended.

Inclure :
- Les objectifs pédagogiques (verbes d'action SMART)
- Le découpage en modules et séquences
- Les méthodes pédagogiques adaptées à chaque module
- Les modalités d'évaluation
- Les ressources recommandées
```

**Pourquoi ça marche :** En précisant le profil, la durée et le format, vous obtenez un plan directement exploitable plutôt qu'une structure générique.

## Prompt 2 : Générer des quiz et QCM

```
Crée 10 questions QCM sur le thème [THÈME] pour évaluer la compréhension d'apprenants de niveau [NIVEAU].

Pour chaque question :
- 4 propositions de réponse (une seule correcte)
- La réponse correcte identifiée
- Une explication courte (2-3 lignes) de pourquoi c'est la bonne réponse

Varie les niveaux cognitifs : mémorisation, compréhension, application, analyse.
```

## Prompt 3 : Adapter un contenu à différents profils

```
J'ai ce contenu pédagogique : [COLLEZ VOTRE TEXTE]

Réécris-le en 3 versions :
1. Pour des débutants complets (pas de jargon, exemples concrets du quotidien)
2. Pour des professionnels intermédiaires (vocabulaire technique acceptable, focus pratique)
3. Pour des experts (dense, références techniques, focus sur les nuances)

Chaque version doit couvrir les mêmes points clés.
```

## Prompt 4 : Créer des études de cas réalistes

```
Crée une étude de cas pédagogique réaliste pour une formation [THÈME] destinée à [PROFIL].

L'étude de cas doit :
- Présenter une situation professionnelle crédible dans le secteur [SECTEUR]
- Comporter un problème à résoudre ou une décision à prendre
- Inclure des données chiffrées et des personnages nommés
- Se terminer par 3 à 5 questions de réflexion pour les apprenants
- Permettre plusieurs approches de résolution valides

Durée de traitement en formation : environ [DURÉE].
```

## Prompt 5 : Rédiger un bilan de compétences individuel

```
Je suis formateur. Sur la base des informations suivantes sur un apprenant, rédige un bilan de compétences personnalisé :

- Nom : [NOM]
- Formation suivie : [FORMATION]
- Durée : [DURÉE]
- Points forts observés : [LISTE]
- Points d'amélioration : [LISTE]
- Résultats aux évaluations : [NOTES/OBSERVATIONS]

Le bilan doit être :
- Bienveillant et constructif
- Précis sur les compétences acquises (avec des verbes d'action)
- Honnête sur les axes de progression
- Court (max 300 mots)
- Rédigé à la troisième personne
```

## Prompt 6 : Préparer des activités brise-glace

```
Propose 5 activités brise-glace pour le début d'une formation de [DURÉE TOTALE] sur [THÈME], avec un groupe de [NOMBRE] personnes qui ne se connaissent pas.

Chaque activité doit :
- Durer entre 5 et 15 minutes
- Ne nécessiter aucun matériel (ou du matériel minimal)
- Favoriser la prise de parole de tous
- Avoir un lien indirect avec le thème de la formation

Précise pour chaque activité : durée, matériel, déroulement pas à pas, variante possible.
```

## Prompt 7 : Transformer un support PowerPoint ennuyeux

```
Voici le contenu de mon slide [NUMÉRO] :

[COLLEZ LE TEXTE DU SLIDE]

Réécris ce contenu pour rendre la slide plus engageante :
1. Un titre accrocheur (max 8 mots)
2. Pas plus de 3 bullets points de max 12 mots chacun
3. Une question ou un chiffre-clé pour accrocher l'attention
4. Un exemple concret en 2 lignes

Garde le même message, rends-le percutant.
```

---

## Prompt 8 : Générer des mises en situation pour la formation présentielle

```
Crée 3 mises en situation pour une formation sur [THÈME] en présentiel, durée totale [DURÉE].

Chaque mise en situation doit :
- Simuler une situation professionnelle réelle
- Impliquer 2 à 4 participants avec des rôles définis
- Avoir un objectif pédagogique clair
- Inclure les instructions pour le formateur (débrief, points à observer)
- Être réalisable en [DURÉE PAR EXERCICE]
```

---

## Prompt 9 : Rédiger une description de formation pour Qualiopi

```
Rédige la description pédagogique de cette formation pour notre dossier Qualiopi :

- Intitulé : [TITRE]
- Durée : [DURÉE]
- Public cible : [PROFIL]
- Prérequis : [LISTE]
- Modalité : [PRÉSENTIEL/DISTANCIEL/BLENDED]
- Objectifs généraux : [LISTE]

La description doit inclure :
1. Les objectifs pédagogiques en verbes d'action mesurables
2. Le programme détaillé par module
3. Les méthodes et moyens pédagogiques
4. Les modalités d'évaluation des acquis
5. Les conditions d'accueil des personnes en situation de handicap

Langage : professionnel, précis, conforme aux exigences du référentiel Qualiopi.
```

---

## Prompt 10 : Analyser les retours d'évaluation à chaud

```
Voici les retours d'évaluation à chaud de ma formation [TITRE] ([NOMBRE] réponses) :

[COLLEZ LES RETOURS]

Analyse ces retours et fournis :
1. Un résumé des points positifs récurrents
2. Les axes d'amélioration les plus cités
3. Une note de satisfaction globale estimée
4. 3 actions concrètes à mettre en place pour la prochaine session
5. Une phrase-bilan à communiquer à ma direction
```
